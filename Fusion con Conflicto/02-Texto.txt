ping <host>: Envía paquetes ICMP ECHO_REQUEST a un host para comprobar la conectividad.
gzip <archivo>: Comprime un archivo usando gzip.
ssh <usuario>@<host>: Conecta a un host remoto utilizando SSH.

scp <origen> <destino>: Copia archivos entre hosts utilizando SSH.

Compresión y Archivado
tar -cvf <archivo.tar> <directorio>: Crea un archivo tar desde un directorio.
tar -xvf <archivo.tar>: Extrae un archivo tar.
gunzip <archivo.gz>: Descomprime un archivo gzip.
