git reset HEAD <nombre-del-archivo>: Quita un archivo del área de preparación.
git reset --soft HEAD~1: Deshace el último commit, manteniendo los cambios en el área de trabajo.
git reset --hard HEAD~1: Deshace el último commit y pierde todos los cambios realizados en el área de trabajo.

mkdir <nombre-del-directorio>: Crea un nuevo directorio.
rm <nombre-del-archivo>: Elimina un archivo.
rm -r <nombre-del-directorio>: Elimina un directorio y su contenido de forma recursiva.
cp <origen> <destino>: Copia un archivo o directorio de origen a destino.
mv <origen> <destino>: Mueve o renombra un archivo o directorio de origen a destino.
